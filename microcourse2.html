FOUNDATIONAL CONTENT

Bring Real-World Creations to Fortnite with Unreal Editor and Photogrammetry

About this Course:

This course is designed to guide creators through the essential steps of using photogrammetry and Unreal Editor for Fortnite to bring real-world elements into the game environment. It consists of comprehensive lessons that provide detailed, step-by-step guidance on building 3D models, importing them into Fortnite, and refining their geometry using advanced tools in Unreal Editor. Each lesson is aimed at helping creators learn practical techniques for creating high-quality models, positioning and scaling them within a virtual landscape, and fine-tuning their structure for optimal presentation and functionality.

Course Modules

MODULE 1: CREATE A 3D MODEL USING POLYCAM

This lesson provides step-by-step guidelines for creating 3D models using the Polycam phone app, and tips for ensuring high-quality models.
